# Design System

## Overview
Design tokens extracted from neon.com.

## Colors
- **Primary** (#ffffff): CTAs, active states, key interactive elements
- **Secondary** (#00e599): Supporting UI, secondary actions
- **Surface** (#ffcce5): Page backgrounds
- **On-surface** (#000000): Primary text

## Typography
- **Headlines**: Inter, regular, 10–80px
- **Body**: esbuild, medium, 56–64px
- **Labels**: GeistMono, regular, 12–20px

## Components
- **Buttons**: Square corners, primary uses #000000 fill, outlined variant available
- **Inputs**: 1px solid border, 0px radius
- **Border radius scale**: 4px, 3.35544e+07px, 10px, 8px
- **Elevation**: uses box shadows for depth

## Do's and Don'ts
- Do use rounded corners consistently across interactive elements
- Do use the primary color sparingly — only for the most important action per screen
- Do maintain 4.5:1 contrast ratio for all body text (WCAG AA)
