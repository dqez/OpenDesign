# Design System

## Overview
Design tokens extracted from gsap.com.

## Colors
- **Secondary** (#000000): Supporting UI, secondary actions
- **Primary** (#ff8709): CTAs, active states, key interactive elements
- **Surface** (#fffce1): Page backgrounds
- **On-surface** (#0e100f): Primary text

## Typography
- **Headlines**: Mori, semi-bold, 11–287.999px
- **Body**: Fraktion Mono, medium, 12.8px

## Components
- **Buttons**: Fully rounded, primary uses #000000 fill, outlined variant available
- **Inputs**: 0px 0px border, 0px radius
- **Border radius scale**: 4px, 100px, 50%, 8px
- **Elevation**: uses box shadows for depth

## Do's and Don'ts
- Do use rounded corners consistently across interactive elements
- Do use the primary color sparingly — only for the most important action per screen
- Do maintain 4.5:1 contrast ratio for all body text (WCAG AA)
